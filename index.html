Case Studies

Detailed analysis of my most successful projects

King House Travel — Performance & Tech SEO

6 weeks

Role: Frontend + Tech SEO • Tools: Lighthouse, PageSpeed, GSC, GA4, JSON-LD

Task

Mobile version slow (Lighthouse 54/100, LCP 4.8s), duplicate meta, no sitemap/FAQ/schema, low CTR for brand and "tours + country".

Solution

  • WebP/AVIF + srcset, loading=lazy, critical CSS inline, defer/async JS, preconnect to fonts.
  • Tech SEO: sitemap.xml, robots.txt, canonical, JSON-LD (Breadcrumb/FAQ/Organization), headings, internal links, titles/descr.

Numbers

Lighthouse: 54 → 93
LCP: 4.8s → 2.1s
CLS: 0.18 → 0.02
INP: 390ms → 160ms
Core Web Vitals (CrUX): 62% → 94%
GSC: clicks +142%, impressions +88%, CTR 1.1% → 3.2%; top-20: 6 → 23
Proof (text)

Lighthouse (mobile) before/after: 54 → 93; LCP 4.8s → 2.1s; CLS 0.18 → 0.02; INP 390ms → 160ms.

PageSpeed (field data): share of good LCP & FID increased, CWV pass 62% → 94%.

GSC: clicks +142%, impressions +88%, CTR 1.1% → 3.2%; in top-20 6 → 23.

Added JSON-LD (Breadcrumb, FAQ, Organization), created sitemap.xml, robots.txt, canonical.

Egypt Tours — content cluster + internal linking

8 weeks

Role: Tech SEO • Tools: GSC, Screaming Frog

  • Cluster linking with internal links + minor performance fixes.

New FAQ blocks marked with FAQPage; internal linking map for the cluster.

Request form: recording 12 leads/month after deploying the cluster.

Tech SEO cleanup — indexing and canonical

30 days

Role: Tech SEO • Tools: GSC, server logs, Screaming Frog

Task

Index bloat (parameters, thin), 404, outdated sitemap, no hreflang (EN/CZ/UK).

Solution

  • Canonical in templates, parameter rules, noindex for thin pages.
  • Sitemap splitting by types, hreflang, 301, fixes for 404/5xx, unified slash policy.

Numbers

Indexed: 1420 → 868 (-39% garbage)
Crawl hits/day: +31%
Coverage errors: -86%
Average position: +3.4
Impressions: +22%
Proof (text)

GSC Coverage: sharp reduction in excluded/errors; increase in valid.

Logs: improvement in crawl budget; more hits on important URLs.

Implemented hreflang, canonicals, 301 redirects; updated sitemap-set.
